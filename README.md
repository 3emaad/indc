## 🌟 نظرة عامة | Overview

مؤشر **INDC** هو أداة تحليل فني شامل مبنية بـ **Pine Script v5** لمنصة TradingView. يجمع المؤشر **16 إشارة مستقلة** من أهم أدوات التحليل الفني ويوفر **قراراً نهائياً للتداول** بناءً على نظام تسجيل نقاط ذكي.

**The INDC indicator** is a comprehensive technical analysis tool built with Pine Script v5 for TradingView. It combines **16 independent signals** from the most important technical analysis tools and provides a **final trading decision** based on a smart scoring system.

### الميزات الرئيسية | Key Features

| الميزة | الوصف |
|--------|--------|
| 🔄 **وايكوف** | تحليل مراحل وأحداث وايكوف كاملة |
| 📦 **Order Blocks** | كشف بلوكات الأوامر الصاعدة والهابطة |
| 📊 **FVG** | الفجوات السعرية وتتبع ملئها |
| 💧 **Liquidity** | مناطق السيولة وكشف Sweeps |
| 🏗️ **Market Structure** | BOS، CHoCH، HH/HL/LH/LL |
| 📈 **11 مؤشر مستقل** | RSI، MACD، BB، EMA، Volume، ATR، Stoch، ADX، Fibonacci، VWAP، Ichimoku |
| ✅ **Dashboard** | لوحة قرار نهائي بنظام نقاط |
| 💬 **Tooltips** | شرح تفصيلي لكل عنصر بالعربي والإنجليزي |
| 🔔 **Alerts** | 14 تنبيه مخصص |

## 🚀 التثبيت | Installation

### الخطوات | Steps

1. **افتح TradingView** — انتقل إلى [tradingview.com](https://www.tradingview.com)
2. **افتح Pine Editor** — اضغط على `Pine Editor` في الشريط السفلي
3. **الصق الكود** — افتح ملف `indicator.pine` من هذا المستودع وانسخ المحتوى كاملاً
4. **أضف للشارت** — اضغط على `Add to chart`
5. **اضبط الإعدادات** — اضغط على ⚙️ لتخصيص الإعدادات حسب احتياجك

```

```

### 1. 🔄 وايكوف | Wyckoff Analysis

تحليل وايكوف هو منهجية تحليل تقني طورها **ريتشارد وايكوف** في مطلع القرن العشرين لفهم سلوك "الأموال الكبيرة" (المؤسسات).

#### مراحل وايكوف | Wyckoff Phases

| المرحلة | الرمز | الوصف |
|---------|-------|--------|
| **Accumulation** 🔄 | تجميع | السوق يتذبذب أفقياً — المؤسسات تشتري خلسة |
| **Markup** 📈 | صعود | السوق يرتفع — المؤسسات تحقق أرباحها |
| **Distribution** ⚠️ | توزيع | السوق يتذبذب أفقياً عند القمة — المؤسسات تبيع |
| **Markdown** 📉 | هبوط | السوق يهبط — العامة يستوعبون الخسائر |

#### أحداث وايكوف | Wyckoff Events

| الحدث | الرمز | الوصف | الإشارة |
|-------|-------|--------|---------|
| **Spring** 🌱 | زنبرك | انخفاض تحت الدعم ثم عودة سريعة | ✅ شراء |
| **Upthrust** 🔺 | دفع فوق المقاومة | ارتفاع فوق المقاومة ثم انعكاس | 🔴 بيع |
| **SOS** 💪 | علامة القوة | حركة صاعدة بحجم مرتفع | ✅ شراء |
| **SOW** ⚠️ | علامة الضعف | حركة هابطة بحجم مرتفع | 🔴 بيع |

### 2. 📦 بلوكات الأوامر | Order Blocks

بلوكات الأوامر هي مناطق سعرية حيث وضعت **المؤسسات والبنوك** أوامر ضخمة، مما يجعلها مناطق دعم ومقاومة مهمة.

#### أنواع بلوكات الأوامر | Types

| النوع | الوصف | اللون |
|-------|--------|-------|
| **Bullish OB** 📦 | آخر شمعة هابطة قبل حركة صاعدة قوية | 🟢 أخضر |
| **Bearish OB** 📦 | آخر شمعة صاعدة قبل حركة هابطة قوية | 🔴 أحمر |

> **كيف تستخدمها:** انتظر عودة السعر لاختبار منطقة البلوك مع تأكيد الإشارة.

---

### 3. 📊 الفجوات السعرية | Fair Value Gaps (FVG)

الفجوات السعرية هي نطاقات سعرية لم تُنفَّذ فيها صفقات بسبب حركة السعر السريعة. السوق غالباً يعود "لملء" هذه الفجوات.

#### أنواع الفجوات | Types

| النوع | الشرط | الإشارة |
|-------|--------|---------|
| **Bullish FVG** ↑ | `low[0] > high[2]` | ✅ شراء عند العودة |
| **Bearish FVG** ↓ | `high[0] < low[2]` | 🔴 بيع عند العودة |

---

### 4. 💧 مناطق السيولة | Liquidity Zones

السيولة هي مجموعات من أوامر **Stop Loss** التي تتراكم فوق القمم وتحت القيعان.

#### مفاهيم السيولة | Concepts

| المفهوم | الوصف |
|---------|--------|
| **BSL** (Buy-Side Liquidity) | سيولة فوق القمم — أوامر Stop Loss للبائعين |
| **SSL** (Sell-Side Liquidity) | سيولة تحت القيعان — أوامر Stop Loss للمشترين |
| **BSL Sweep** 💧 | اختراق فوق القمة ثم إغلاق تحتها = **إشارة بيع** |
| **SSL Sweep** 💧 | اختراق تحت القاع ثم إغلاق فوقه = **إشارة شراء** |

---

### 5. 🏗️ هيكل السوق | Market Structure

هيكل السوق يحدد الاتجاه العام من خلال تحليل القمم والقيعان المتتالية.

#### عناصر هيكل السوق | Elements

| العنصر | الوصف | الإشارة |
|--------|--------|---------|
| **HH** (Higher High) | قمة أعلى من السابقة | ↗️ صاعد |
| **HL** (Higher Low) | قاع أعلى من السابق | ↗️ صاعد |
| **LH** (Lower High) | قمة أدنى من السابقة | ↘️ هابط |
| **LL** (Lower Low) | قاع أدنى من السابق | ↘️ هابط |
| **BOS ▲** | كسر هيكل صاعد | ✅ شراء |
| **BOS ▼** | كسر هيكل هابط | 🔴 بيع |
| **CHoCH ▲** | تغيير طابع السوق صاعد | ✅✅ شراء قوي |
| **CHoCH ▼** | تغيير طابع السوق هابط | 🔴🔴 بيع قوي |

### 6. 📈 المؤشرات الفنية | Technical Indicators

كل مؤشر يعمل **بشكل مستقل** ويعطي إشارته الخاصة:

#### RSI — مؤشر القوة النسبية
- **تشبع بيعي (< 30):** ✅ إشارة شراء
- **تشبع شرائي (> 70):** 🔴 إشارة بيع
- **محايد (30-70):** ➡️ لا إشارة

#### MACD
- **MACD فوق Signal:** ✅ إشارة شراء
- **MACD تحت Signal:** 🔴 إشارة بيع

#### Bollinger Bands
- **السعر تحت النطاق السفلي:** ✅ إشارة شراء
- **السعر فوق النطاق العلوي:** 🔴 إشارة بيع

#### EMA (20 / 50 / 200)
- **Golden Cross (50 يتجاوز 200 صعوداً):** ✅ إشارة شراء قوية
- **Death Cross (50 يتجاوز 200 هبوطاً):** 🔴 إشارة بيع قوية
- **السعر فوق الثلاثة متوسطات مرتبة:** ✅ إشارة شراء
- **السعر تحت الثلاثة متوسطات مرتبة:** 🔴 إشارة بيع

#### Volume — حجم التداول
- **حجم > 1.5x المتوسط مع صعود:** ✅ إشارة شراء
- **حجم > 1.5x المتوسط مع هبوط:** 🔴 إشارة بيع

#### ATR — متوسط المدى الحقيقي
- **ATR متزايد مع سعر فوق EMA50:** ✅ زخم صاعد
- **ATR متزايد مع سعر تحت EMA50:** 🔴 زخم هابط

#### Stochastic RSI
- **K < 20:** ✅ تشبع بيعي — إشارة شراء
- **K > 80:** 🔴 تشبع شرائي — إشارة بيع

#### ADX — قوة الاتجاه
- **ADX > 25 و DI+ > DI-:** ✅ اتجاه صاعد قوي
- **ADX > 25 و DI- > DI+:** 🔴 اتجاه هابط قوي
- **ADX < 25:** ➡️ لا اتجاه واضح

#### Fibonacci — مستويات تلقائية
- **السعر قرب 61.8% / 50% / 78.6% في اتجاه صاعد:** ✅ دعم فيبو
- **السعر قرب 23.6% / 38.2% في اتجاه هابط:** 🔴 مقاومة فيبو

#### VWAP — متوسط السعر الموزون بالحجم
- **السعر فوق VWAP:** ✅ إشارة شراء
- **السعر تحت VWAP:** 🔴 إشارة بيع

#### Ichimoku Cloud — سحابة إيشيموكو
- **السعر فوق السحابة:** ✅ إشارة شراء
- **السعر تحت السحابة:** 🔴 إشارة بيع
- **السعر داخل السحابة:** ➡️ محايد

### 7. ✅ القرار النهائي | Final Decision Dashboard

لوحة **Dashboard** تظهر في زاوية الشارت وتجمع نتائج جميع المؤشرات الـ16.

#### نظام القرار | Decision System

| القرار | النسبة | المعنى |
|--------|--------|--------|
| 🚀 **شراء قوي \| STRONG BUY** | ≥ 70% إشارات شراء | فرصة شراء ممتازة |
| ✅ **شراء \| BUY** | 55-70% إشارات شراء | فرصة شراء جيدة |
| ⚖️ **محايد \| NEUTRAL** | 45-55% | لا توجد إشارة واضحة |
| 🔴 **بيع \| SELL** | 55-70% إشارات بيع | فرصة بيع جيدة |
| 🔻 **بيع قوي \| STRONG SELL** | ≥ 70% إشارات بيع | فرصة بيع ممتازة |

#### محتوى الجدول | Table Contents

- **العمود 1:** اسم المؤشر
- **العمود 2:** القيمة الحالية + التفاصيل
- **العمود 3:** الإشارة (✅ شراء / 🔴 بيع / ➡️ محايد)
- **الصف الأول:** القرار النهائي + نسبة الثقة
- **الصفوف الأخيرة:** ملخص نقاط الشراء والبيع والمحايد

## 🏆 نظام التسجيل | Scoring System

```
إجمالي المؤشرات: 16

كل مؤشر يعطي:
  +1  →  إشارة شراء
   0  →  محايد
  -1  →  إشارة بيع

نسبة الشراء  = (عدد إشارات الشراء  / 16) × 100
نسبة البيع   = (عدد إشارات البيع   / 16) × 100

القرار النهائي:
  🚀 شراء قوي  →  نسبة شراء ≥ 70%
  ✅ شراء       →  نسبة شراء 55-70%
  ⚖️ محايد     →  لا شراء ولا بيع واضح
  🔴 بيع        →  نسبة بيع 55-70%
  🔻 بيع قوي   →  نسبة بيع ≥ 70%
```
